# 八皇后问题

## 问题描述

如何能够在 8×8 的国际象棋棋盘上放置八个皇后，使得任何一个皇后都无法直接吃掉其他的皇后？  
即要求任两个皇后都不能处于同一条横行、纵行或对角线上。
